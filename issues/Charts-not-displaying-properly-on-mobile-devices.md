# Charts not displaying properly on mobile devices

### Description
The charts are not visible for all categories when viewed on mobile devices. This issue needs to address mobile responsiveness and ensure that charts adjust correctly according to different screen sizes.

### Steps to reproduce
1. Open the website on a mobile device or resize the browser to simulate a mobile view.
2. Navigate to the charts section.
3. Observe that not all charts are visible.

### Proposed Solution
Investigate the responsive design settings for the chart rendering library in use and adjust CSS or JavaScript as necessary.

### Environment
Mobile devices and small browser windows, across multiple browsers.
